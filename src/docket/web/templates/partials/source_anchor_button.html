{# Source-anchor adaptive button — spec §6.4. Renders a "View Source" link that adapts to the captured anchor level: 1. PDF + bbox + page → PDF page link (region marker emitted; PDF viewers honour ``#page=N`` natively, no JS). 2. PDF + page → PDF page link. 3. PDF (bare) → bare PDF URL. 4. HTML + anchor → HTML page with ``#anchor`` fragment. 5. Video + timestamp → ``?t=NNN`` deep link, label includes the formatted timestamp via the ``format_timestamp`` filter. 6. Any ``anchor.url`` → bare "View Source" fallback. 7. ``data_quality == 'no_text_layer'`` → unavailable label, plus an admin-only link to the data-debt queue. 8. Otherwise → no markup. Every external link uses ``target="_blank"`` and ``rel="noopener noreferrer"`` (matches the convention from E1's ``_source_link_stub.html`` and E3's engagement strip). URL scheme allowlist (mirrors :file:`_source_link_stub.html`): only ``http://`` and ``https://`` URLs produce a clickable link. Anything else — ``javascript:``, ``data:``, protocol-relative ``//host/path``, bare schemes — is rejected silently. The PDF/HTML/video branches all funnel through the same ``_safe`` guard so a malicious anchor can't bypass the check by claiming a specific ``type``. Defensive against malformed input: ``anchor`` is coerced via ``or {}`` so a missing/None ``source_anchor`` won't blow up. If the stored value is a non-mapping (string, list, etc.) ``.type`` / ``.url`` lookups return Undefined and silently fail every branch, landing on the no-render path — no traceback in production. Inputs (via implicit ``item`` context, dispatcher passes the row): - ``item.source_anchor`` — JSONB; one of: {type: 'pdf', url, page?, bbox?} {type: 'html', url, anchor?} {type: 'video', url, timestamp_seconds?} {url} (bare fallback) - ``item.data_quality`` — used only for the OCR-needed branch. - ``item.id`` — needed for the admin queue link. - ``session.admin_user`` — truthy iff the requester is logged in as an admin (codebase convention; spec references ``g.user.is_admin``, the codebase uses session-based auth). #} {%- set anchor = item.source_anchor or {} -%} {%- set _url = anchor.url if anchor is mapping else None -%} {%- set _is_safe = _url and (_url.startswith('http://') or _url.startswith('https://')) -%} {% if anchor is mapping and anchor.type == 'pdf' and _is_safe %} {% if anchor.bbox and anchor.page %} View Source: PDF page {{ anchor.page | int }} (region) → {% elif anchor.page %} View Source: PDF page {{ anchor.page | int }} → {% else %} View Source: PDF → {% endif %} {% elif anchor is mapping and anchor.type == 'html' and anchor.anchor and _is_safe %} View Source: agenda item → {% elif anchor is mapping and anchor.type == 'video' and anchor.timestamp_seconds and _is_safe %} View Source: video at {{ anchor.timestamp_seconds | format_timestamp }} → {% elif _is_safe %} View Source → {% elif item.data_quality == 'no_text_layer' %} Source needs OCR {% if session.admin_user %} [admin queue] {% endif %} {% endif %}
